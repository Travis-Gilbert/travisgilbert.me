# Object Exist, Nodes Happen. 

This is an object-oriented, spatial, and temporal… note-taking app? I am still figure it out tbh. It’s complicated thing meant to make complexity easy to handle. I’ve always had a poor memory and time-blindness. When I was going through challenging time, I need to compensate the issues caused by my ADHD to be sharper than I was I built this app. I hope you find it useful. Travis, [https://travisgilbert.me/commonplace](https://travisgilbert.me/commonplace) P.S. I built a 7 stage connection engine from PyTorch, Spacy, SciPy, Tesseract OCR, KGE and more The engine powers the app. I’m not a prideful person (I think), but I am proud of this. I aspire to someday build an app powerful enough to help compensate for my near-crippling, OCD.
